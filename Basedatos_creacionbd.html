CREACION DE BASE DE DATOS

Crear una base de datos es un proceso muy sencillo el cual se puede ejecutar en PhpMyadmin de dos diferentes formas, una es por la interfaz del PhpMyadmin y otro es por consola, en caso de ejecutarse por consola, que en la mayoria de los casos es la mas optima para un desarrollador, se debe ejecutar el siguiente codigo: 

    <mysql> CREATE DATABASE unadistas >
Donde "unadistas" es el nombre de la base de datos

La sintaxis basica de mysql se debe tener en cuenta para realizar todo tipo de procesos en la gestion de una base de datos, aqui se presentan los comando basicos que luego se explicaran en el video , que explica el tema de manera practica.

Crear una Tabla:
    <create table estudiantes(nombre varchar(20)) >
Donde "estudiantes" es el nombre de la tabla y lo que esta dentro del parentesis el nombre del campo o campos y su respectivo tipo de dato

Seleccionar Registro:

<select *from estudiantes>
Donde "estudiantes" es el nombre de la tabla y el asterisco(*) equivale a seleccionar todos los datos

Insertar Registro:

<Insert into tb estudiantes(nombre) values("Pedro")>
Donde "estudiantes" es el nombre de la tabla y values los valores el valor o valores a insertar dependiendo de los campos

Actualizar Registro:

<Upate tb estudiantes set nombre="JUAN" where nombre="Pedro" >
Donde "estudiantes" es el nombre de la tabla, set y el campo que se va a actualizar, y where es donde se aplica el cambio

Eliminar Registro:

<Delete from tb estudiantes  where nombre="Pedro" >
Donde "estudiantes" es el nombre de la tabla, y where es donde se aplica la eliminacion del registro
