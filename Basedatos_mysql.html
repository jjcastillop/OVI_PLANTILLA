    BASE DE DATOS MYSQL

    MySQL es un sistema de gestión de base de datos relacional (RDBMS) de código abierto, basado en lenguaje de consulta estructurado (SQL). Las siglas SQL significa en inglés Structured Query Language; en español lenguaje de consulta estructurada, es un lenguaje específico del dominio que da acceso a un sistema de gestión de bases de datos relacionales que permite especificar diversos tipos de operaciones en ellos.

    Actualmente MySql es uno de los gestores de base de datos mas utilizados en el mundo, al ser un gestor libre tiene como beneficio el completo soporte comunitario que existe a lo largo de internet, ademas de su facil adaptabilidad a los lenguajes de programacion mas comunes y su compatibilidad.
